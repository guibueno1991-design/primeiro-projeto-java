# 🏦 Dev-Bank - Sistema Bancário em Java

Este é um sistema bancário simples executado via console, desenvolvido para praticar e consolidar os conceitos fundamentais da linguagem Java.

## 🚀 Funcionalidades
* **Consulta de Saldo:** Exibe o saldo atualizado do cliente de forma instantânea.
* **Depósito:** Permite somar novos valores ao saldo atual.
* **Saque:** Deduz valores do saldo, contendo uma validação de segurança para impedir saldos negativos.

## 🛠️ Tecnologias Utilizadas
* **Java** (Lógica principal e controle de fluxos)
* **Git & GitHub** (Controle de versão e hospedagem do código)
